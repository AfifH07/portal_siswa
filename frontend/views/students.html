📋 Petunjuk Import Data Siswa

Download template Excel yang sudah disiapkan dengan format yang benar:

  • NISN - Nomor Induk Siswa Nasional (wajib, unik)
  • NIS - Nomor Induk Siswa lokal, 7 digit (wajib, unik)
  • Nama - Nama lengkap siswa (wajib)
  • Jenis Kelamin - L (Laki-laki) / P (Perempuan)
  • Kelas - Contoh: X A, XI B, XII C
  • Program - Reguler / Tahfidz / Khusus
  • Catatan - Catatan tambahan (opsional)

Tips: Untuk NISN/NIS yang diawali angka 0, tambahkan tanda petik (') di depan. Contoh: '0069028700
